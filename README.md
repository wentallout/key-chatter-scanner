# KeyChatterScanner

This is a simple but very useful tool to look for keychattering in all of the keys in the keyboard

It will look like a typical key tester. But it will highlight red key that are chattering and list them in a table below, user can then fix each key and tick the checkbox to mark that key as fix

Minimal, friendly UI UX, prob add sounds if key chattering is detected
